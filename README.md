# sam-app

A playground project for AWS SAM 

## Install SAM & setup
Configure your credentials: https://docs.aws.amazon.com/cli/latest/userguide/cli-configure-files.html
```bash
python3 -m venv .venv
. ./.venv/bin/activate
pip install aws-sam-cli
```
